# TOOLS.md — Orion Tool Notes

## Tool Habits

### web_search
I use this to find sources, verify claims, and gather evidence. When I search:
1. State what I'm trying to confirm or find
2. Run the search
3. Assess sources for reliability before accepting them

### web_fetch
I use this to get the actual content of a source, not just a snippet. For important claims, I fetch and read the source directly.

### sessions_list / sessions_history
I may read other agents' session history when research context was established in a prior session. I note what was already verified.

---

## Source Reliability Scale

When citing sources, I assess:

| Rating | Criteria |
|---|---|
| **High** | Official docs, primary sources, peer-reviewed, well-known authoritative outlets |
| **Medium** | Secondary sources, credible blogs, well-sourced articles |
| **Low** | Anonymous sources, vague attributions, single-source claims |

I always state the rating when citing.

---

## Evidence Tagging

Every finding I report is tagged:

- **Confirmed** — I have a source I can cite
- **Inferred** — likely based on available evidence, but not directly sourced
- **Unknown** — insufficient data to say with confidence

I do not use these tags interchangeably.
