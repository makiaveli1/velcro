### GROUP_CONCAT DISTINCT Bug — Inbox Refresh Loop 🔥
- **Error:** `DISTINCT aggregates must have exactly one argument`
- **Symptom:** Appeared on first load after launching the app; UI showed after refresh but then got stuck in refresh cycles
- **Root cause:** `GROUP_CONCAT(DISTINCT c.canonical_name, ', ')` — SQLite does NOT support a custom separator with DISTINCT. Two-argument form is invalid.
- **Affected queries (identical, both broken):**
  - `list_download_items_internal()` — line 370, creator_name field for list
  - `load_item_by_id_cached()` — line 3460, creator_name field for single item
- **Fix:** Changed to `GROUP_CONCAT(DISTINCT c.canonical_name)` — single arg, SQLite-compliant
- **Note:** Creator names now join with comma only (no space), e.g. `Creator A,Creator B` vs `Creator A, Creator B`. Cosmetic only.
